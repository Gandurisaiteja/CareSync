# CareSync – Healthcare Management System

CareSync is a full-stack healthcare management system developed using ReactJS, Node.js, Express.js, and SQLite.  
The application supports multiple user roles including Admin, Doctor, Patient, and Receptionist.

---

# Features

## Authentication & Authorization
- User Registration & Login
- JWT Authentication
- Role-based access control

## Admin Dashboard
- View analytics
- Manage appointments
- Manage doctors and patients
  

## Doctor Features
- Manage appointments
- Add prescriptions
- View patient medical history

## Patient Features
- Book appointments
- View prescriptions
- View appointment history
- Medical records access

## Receptionist Features
- Manage appointments
- Update appointment status

## Additional Features
- Responsive dashboard UI
- Sidebar navigation
- Search & filtering
- Loading states
- Form validations
- Error handling
- Protected routes
- SQLite database integration
